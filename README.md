# 📊 Marketing Funnel & Conversion Dashboard (Excel)

## 🔷 Project Overview

This project analyzes marketing funnel performance using Excel. It focuses on understanding how users move through different stages — from initial contact (traffic/leads) to final conversion (customers).

The dashboard provides insights into conversion rates, drop-off points, and performance across channels and time periods.

---

## 🎯 Objectives

* Clean and structure raw marketing data
* Analyze funnel performance
* Calculate:

  * Traffic-to-Lead Conversion
  * Lead-to-Customer Conversion
* Identify drop-off points in the funnel
* Compare performance across:

  * Channels
  * Campaigns
  * Time (monthly trends)
* Provide actionable business recommendations

---

## 🧰 Tools Used

* Microsoft Excel

  * Pivot Tables
  * Pivot Charts
  * Slicers (Interactive Filters)
  * Formulas & Functions

---

## 📂 Dataset

* Bank Marketing Dataset (CSV format)
* Contains customer interaction and campaign data

---

## 🔧 Data Preparation

* Converted raw data into Excel Table
* Created calculated columns:

  * Traffic (all rows = 1)
  * Leads (all rows = 1)
  * Customers (based on "yes/no" response)
* Cleaned and standardized data for analysis

---

## 📈 Key Metrics

* **Traffic → Lead Conversion Rate**
* **Lead → Customer Conversion Rate**
* **Drop-off Rate**
* Total Leads & Customers

---

## 📊 Dashboard Features

* Funnel Visualization (Traffic → Leads → Customers)
* KPI Cards (Total Leads, Customers, Conversion Rate)
* Channel Performance Analysis
* Monthly Trend Analysis
* Interactive Filters (Slicers):

  * Month
  * Job
  * Contact Type

---

## 🔍 Key Insights

* High drop-off observed between Leads and Customers
* Certain channels (e.g., cellular) show better conversion performance
* Conversion trends vary across months
* Majority of leads do not convert, indicating optimization opportunity

---

## 💡 Recommendations

* Focus on high-performing channels to improve ROI
* Improve follow-up strategies for leads
* Optimize campaign timing based on monthly trends
* Personalize targeting based on customer segments
